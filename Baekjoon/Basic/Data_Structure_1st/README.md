### 풀이 전 알아야 할 개념

#### 스택

스택은 LIFO 방식으로 데이터를 처리하는 자료구조입니다.  
대표적인 연산은 push, pop, top, empty, size 가 있습니다.  
스택은 array, linked list를 이용해 구현할 수 있으며, STL에 구현되어 있는 stack을 사용하는 쪽을 더욱 추천합니다.  
스택에 대한 구체적인 구현 방식은 [스택](./Stack) 을 참고하시면 됩니다.
